
# ssb-patchql-client

An example app that has feeds / threads / posts etc.

## how

### install

```
git clone https://github.com/pietgeursen/ssb-patchql-client
cd 
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
